## Decisions

- **[DECISION A] Built-in strip pass, not external `tsc`** (settles Q1). The `tsc`
  route reintroduces the Node/`tsc` dependency the track exists to remove and
  cannot run in-browser; the wasm slice needs JS produced in-process.
- **[DECISION B] oxc, in a dedicated crate.** Pure-Rust, wasm-safe, purpose-built;
  isolated so `bynk-emit`/`bynk-ide`/`bynk-lsp` stay lean. Hand-rolling rejected
  (the emitted surface's `:`/`<â€¦>`/`as`/`type` ambiguities need a real parser).
- **[DECISION C] Pure type-stripping semantics.** Preserve all value imports;
  erase only type syntax â€” byte-faithful to "emit-then-strip".
- **[DECISION D] Post-emit step in `bynkc`, emitter stays TS-only.** Keeps the
  emitter single-purpose and oxc downstream of it.
- **[DECISION E] Version â€” patch v0.108.1** (track-internal increment).

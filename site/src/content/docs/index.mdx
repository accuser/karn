A statically typed language for backend services in which the architecture —
contexts, services, agents, and the capabilities that flow between them — is part
of the language and checked by the compiler. It compiles to typed TypeScript and
runs on Cloudflare Workers.

```bynk
context shortener

service api from http {
  on GET("/links/:code") by Visitor (code: String) -> Effect[HttpResult[String]] {
    NotFound
  }
}
```

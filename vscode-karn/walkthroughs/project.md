## Create a project

A Karn project is a folder with a `karn.toml` manifest and a `src/` directory.
The manifest marks the project root so the language server can resolve names
across files.

Run **Karn: New Project** (the button below, or the Command Palette) to scaffold
`karn.toml` plus a starter `context` under `src/`.

Already inside a project? Use **Karn: New Context** to add another `context` or
`commons` file.

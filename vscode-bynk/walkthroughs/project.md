## Create a project

A Bynk project is a folder with a `bynk.toml` manifest and a `src/` directory.
The manifest marks the project root so the language server can resolve names
across files.

Run **Bynk: New Project** (the button below, or the Command Palette) to scaffold
`bynk.toml` plus a starter `context` under `src/`.

Already inside a project? Use **Bynk: New Context** to add another `context` or
`commons` file.
